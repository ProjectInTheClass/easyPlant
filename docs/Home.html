EasyPlant의 홈 화면에서 제공하는 기능들을 소개합니다.

캘린더 기능

캘린더의 날짜를 클릭하면 아래의 식물 목록 창에 해당 날짜마다 물을 주어야하는 식물 목록들이 나타납니다.

식물 별로 설정된 커스텀 컬러를 통해 캘린더만 보고도 언제 물을 주어야 하는지 간편하게 확인할 수 있습니다.

나의 레벨 기능

일반인부터 최종 단계인 이지 플랜터로 구성되는 총 7개의 레벨 업 기능을 제공합니다.

easyPlant를 통해 식물을 키운 기간, 식물의 행복도, 식물의 개수를 통해 사용자의 레벨을 산출합니다.

식물의 행복도는 최근 물을 준 10번의 기록을 통해 산출됩니다.

식물의 행복도에 따라 홈 화면에 나타나는 식물 그림이 행복하거나, 우울해질 수 있습니다.

물 주기 기능

식물 목록 창에 나타나는 물뿌리개 아이콘을 클릭하여 식물에게 물을 줄 수 있습니다. 물을 제대로 주었다면 다음날 모든 정보가 업데이트되며, 식물의 행복도가 올라갑니다.
